# Lane Segmentation

Учебный проект по сегментации дорожных полос на изображениях дороги. Для работы использовался датасет TuSimple и нейросеть U-Net на TensorFlow/Keras.

## Что мы делали

В ходе проекта был подготовлен полный путь от датасета до готовой модели сегментации дорожных полос.

Мы разобрали структуру датасета TuSimple, где изображения дороги хранятся отдельно, а координаты дорожных полос находятся в JSON-разметке. После этого был написан код, который преобразует координаты полос в бинарные маски. Эти маски используются как правильные ответы для обучения нейросети.

Для сегментации была выбрана архитектура U-Net. Модель обучалась определять пиксели дорожных полос на изображении. Для оценки качества были добавлены метрики Dice и IoU, потому что для сегментации они информативнее обычной accuracy.

Обучение выполнялось в Google Colab с использованием GPU. Сначала был сделан тестовый запуск на 200 изображениях, чтобы проверить, что весь pipeline работает правильно. Затем была обучена рабочая demo-модель на 1000 изображениях.

После обучения модель была сохранена в формате Keras. В репозитории лежит обученная модель:

```text
models/unet_tusimple_1000.keras
```

Также был сделан локальный интерфейс . Через него можно выбрать дорожное изображение и визуально посмотреть результат работы модели: исходное фото, предсказанную маску и наложение полос поверх дороги.

## Что получилось


- обработку разметки TuSimple;
- генерацию масок дорожных полос;
- U-Net модель для сегментации;
- обучение в Google Colab;
- метрики Dice и IoU;
- обученную модель на 1000 изображениях;
- пример результата сегментации;
- локальный интерфейс для проверки изображений.
